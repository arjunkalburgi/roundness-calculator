How to calculate
the roundness?

When you have rounded boxes inside each other, the flow between them will be unnatural unless the roundness is perfect. simply substract the side distance from the larger corner radius.

            
.outer {
    border-radius: 64px;
    padding: 40px;
}
    
.inner {
    border-radius: 24px;
}
